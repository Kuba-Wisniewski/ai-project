## How To Run

### 1. Create a Virtual Environment

```bash
python -m venv .venv
# or
python3 -m venv .venv
```

---

### 2. Activate the Virtual Environment

* **Linux / macOS:**

```bash
source .venv/bin/activate
```

* **Windows (PowerShell):**

```powershell
.venv\Scripts\Activate.ps1
```

* **Windows (CMD):**

```cmd
.venv\Scripts\activate.bat
```

---

### 3. Install requirements

```sh
pip install -r requirements.txt
```

---

### 4. Run main.py file

```sh
python src/main.py
```
